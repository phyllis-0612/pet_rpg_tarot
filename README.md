# Pet RPG Tarot Generator 🐾🔮

这是一个轻量级的 Python 趣味命令行工具。它结合了塔罗牌的随机抽取与 RPG 游戏的角色职业设定，可以一键为指定角色生成今日的专属运势与战斗属性。

## 功能特性
- 🎲 随机抽取经典大阿尔卡纳塔罗牌。
- ⚔️ 分配随机的 RPG 幻想职业（如占星学者、幻影刺客等）。
- 🔥 生成 50-100 的随机初始战斗力数值。

## 如何运行
1. 确保你的电脑上安装了 Python 3 运行环境。
2. 克隆或下载本仓库代码。
3. 在终端执行以下命令：
   ```bash
   python pet_rpg_tarot.py
